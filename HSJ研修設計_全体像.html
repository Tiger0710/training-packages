営業職

※当社の営業職は技術知識を前提とした営業活動が求められる。技術を理解した上で客先と会話し、提案・調整を行う。

スキル担当〜主任課長代理課長
技術理解自社の施工内容・工法・使用機材を理解し、客先に説明できる客先の技術的な要望を理解し、社内の技術部門と正確に橋渡しできる技術動向を把握し、客先に対し技術的な裏付けを持った提案ができる
信頼構築期日・約束を守り、誠実な対応ができる客先の課題やニーズを把握し、的確に応えられる客先の経営課題を理解し、パートナーとして関係を築ける
提案・交渉見積依頼に対して正確に対応できる客先の要望を整理し、自社に有利な条件で調整できる価格・工期・仕様について客先と対等に交渉できる
計数管理実行予算の構造を理解し、予算内で業務を進められる見積作成から請求まで一連の数字を管理できる部門の売上・粗利を予測し、予実差異を分析できる
案件管理担当案件の工程・進捗を把握し、社内に共有できる案件の工程・コスト・リスクを総合的に判断し、社内外と調整できる複数案件を俯瞰し、優先順位とリソース配分を判断できる
育成自分の業務を言語化し、後輩に伝えられる後輩にOJTで体系的に指導できる部下の強み・課題を見極め、育成計画を設計できる

技術職

スキル担当〜主任課長代理課長

工程管理工程表を読み、自分の作業を段取りできる一物件の工程を管理し、遅延リスクを予測できる複数現場のリソース配分と優先順位を判断できる
安全管理法令・施工GL・クライアントルールを理解し遵守できる現場のリスクを評価し、適切な安全対策を実行できる安品と連携し、現場全体の安全管理体制を構築できる
原価管理工事原価の構造（材料・労務・外注）を理解している実行予算を作成し、原価を意識して行動できる外注交渉・予算評価を含め、部門の原価を管理できる
問題解決異常に気づき、正確に報告できるトラブルの原因を切り分け、対処できる再発防止の仕組みを設計し、組織に定着させられる
育成自分の作業手順を後輩に説明できる後輩の技量に合わせた指導ができる部下の育成計画を設計し、成長を管理できる

3安全教育プログラム

安全教育は研修体系の1項目ではなく、独立した継続プログラムとして運用する。 電気通信工事は高所作業・感電リスクを伴う。安全教育の不備は元請からの信頼喪失に直結する。
  • 入社時教育（必須・約27時間） 労働安全衛生法の基礎 / KY活動 / 保護具の正しい使い方 / 電気安全 / 高所作業の安全 / 事故事例研究 / 緊急時対応 / 車両安全
  • 継続教育（年間サイクル） 毎日の朝礼KY活動 → 月1回安全ミーティング → 四半期パトロール振り返り → 年1回安全大会・フルハーネス再訓練 → 季節別対策（熱中症・冬季）
  • 法定特別教育 フルハーネス型墜落制止用器具（入社時） / 低圧電気取扱い（入社時） / 酸素欠乏危険作業（必要時） / 職長・安全衛生責任者教育（主任昇格時）

4コア技術研修プログラム

  • 基礎技術（担当〜主任向け・約44時間） LAN基礎 / IP基礎 / 光ファイバー基礎 / UTPケーブル成端 / 融着接続 / 測定器の使い方 / 基地局工事の基礎 / 図面の読み方 / 施工報告書

9ブラザー・シスター制度

新入社員・中途入社者に対し、年齢の近い先輩社員を「ブラザー/シスター」として配置する。

制度概要

  • 対象: 入社1〜2年目の社員
  • 担当者の条件: 入社3〜7年目、同部門所属
  • 担当人数: 1名あたり新人1名
  • 期間: 入社後1年間

役割と運用

  • 役割: 実務指導 + 精神面のフォロー + 組織への馴染み支援
  • 面談: 週1回の定期面談（30分）+ 日常的なフォロー
  • 上長の関与: 月1回、担当者・新人・上長の三者面談
  • 事前研修: 任命前に「指導者研修」（半日）を受講
