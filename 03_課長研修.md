# 課長研修パッケージ

## 研修の目的

課長は部門運営の要であり、経営層の方針を現場に落とし込み、成果を出す中核的マネジメント層である。本研修では、組織マネジメント・プロジェクト管理・予算管理・法令遵守・渉外力・戦略思考を体系的に習得し、部門の業績向上と人材育成を両立できる課長を育成する。

**対象者:** 課長職（新任課長および既任課長のリフレッシュ研修）
**研修期間:** 年間12回（月1回・各回終日 or 半日）+ 集中合宿1回
**形式:** 座学講義、ケーススタディ、グループディスカッション、ロールプレイ、外部講師招聘、現場実習

## 2. マネジメント研修

### 2-1. 組織マネジメントの原則（5月）

**目的:** 課長として部門全体を俯瞰し、組織成果を最大化するマネジメントの基盤を築く

| 項目 | 内容 |
|---|---|
| **組織設計の基本** | 機能別組織・プロジェクト型組織の理解、電気通信工事会社における組織構造の特徴 |
| **PDCAとOODAループ** | 現場レベルのPDCA運用、変化の激しい現場でのOODA活用 |
| **権限委譲と報告体制** | 主任・係長への適切な権限委譲、エスカレーションルールの設計 |
| **チームビルディング** | 施工班・設計班・営業班の連携強化、多能工化の推進 |
| **会議体の設計・運営** | 週次進捗会議・月次部門会議の効率的な運営方法 |

**演習:**
- ケーススタディ: 「工期遅延が発生した大型プロジェクトで、部門内のリソース再配分をどう行うか」
- グループワーク: 自部門の組織課題を洗い出し、改善策を発表

---

### 2-2. 部下育成・評価面談スキル（6月）

**目的:** 部下の成長を支援し、公正な評価を通じて組織力を高める

| 項目 | 内容 |
|---|---|
| **育成計画の策定** | 技術者・施工管理者のキャリアパス設計、資格取得支援計画 |
| **OJTの体系化** | 現場同行指導、段階的な責任付与、振り返りミーティングの手法 |
| **1on1ミーティング** | 傾聴スキル、コーチング質問技法、フィードバックの仕方（SBI法） |
| **評価面談の実施** | 目標設定面談（SMART目標）、中間面談、期末評価面談の進め方 |
| **評価の公平性** | 評価バイアス（ハロー効果・中心化傾向等）の理解と対策 |
| **困難な場面の対処** | 低評価者への伝え方、成長意欲が低い部下へのアプローチ |

**演習:**
- ロールプレイ: 評価面談（高評価者/低評価者の2パターン）
- ロールプレイ: 若手技術者がミスを繰り返す場面での指導面談
- 演習: 自部門メンバーの育成計画シート作成

**推奨資格:**
- 国家資格キャリアコンサルタント関連知識（参考レベル）

## 4. 予算・収益管理（8月・1月）

### 4-2. 原価管理と利益改善

| 項目 | 内容 |
|---|---|
| **工事原価の構造** | 材料費30%・外注費40%・労務費20%・経費10%（目安）の内訳把握 |
| **実行予算管理** | 受注時の実行予算作成、工事進行に伴う原価追跡 |
| **粗利率管理** | 案件別粗利率のモニタリング、赤字案件の早期発見と対策 |
| **コストダウン手法** | VE（バリューエンジニアリング）、購買集約、工法改善 |
| **完工精算** | 工事完了後の精算プロセス、振り返りによるナレッジ蓄積 |

| 利益改善の着眼点 | 具体策 |
|---|---|
| 材料費削減 | メーカー直仕入れ、他現場との共同購買、余剰材料の転用 |
| 外注費最適化 | 協力会社の評価・ランク付け、長期契約による単価交渉 |
| 労務生産性向上 | 多能工育成、作業手順標準化、ICTツール活用 |
| 追加工事の獲得 | 現場での追加ニーズ発見力、適切な見積提出のスピード |

## 8. 評価基準

### 8-1. マネジメント能力 評価指標

課長のマネジメント能力を以下の6領域・18項目で評価する。各項目を5段階（1: 未達 ～ 5: 卓越）で評価し、半期ごとに上長が面談を通じて評価する。

#### 評価シート

| 評価領域 | 評価項目 | 評価基準（レベル3: 標準の例） | 配点 |
|---|---|---|:---:|
| **組織運営** | 目標設定・進捗管理 | 部門目標をメンバーに落とし込み、月次で進捗確認を行っている | 5 |
| | 権限委譲・育成 | 主任以上に適切に権限委譲し、判断力を育てている | 5 |
| | チームワーク醸成 | 部門内のコミュニケーションが活発で、協力体制が機能している | 5 |
| **業績管理** | 売上目標達成 | 部門売上目標の達成率95%以上 | 5 |
| | 利益管理 | 部門粗利率が目標値以上を維持している | 5 |
| | 新規開拓 | 新規顧客または新規案件の獲得に向けた活動を計画的に行っている | 5 |
| **プロジェクト管理** | 工程管理 | 担当プロジェクトの工期遵守率90%以上 | 5 |
| | 品質管理 | 重大な品質事故ゼロ、手直し率3%以下 | 5 |
| | 安全管理 | 重大災害ゼロ、安全パトロール月1回以上実施 | 5 |
| **人材育成** | 部下の成長支援 | 部下の資格取得・スキルアップを計画的に支援している | 5 |
