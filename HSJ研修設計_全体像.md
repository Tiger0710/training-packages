# HSJ 研修設計 全体像

## 前提

- 業態: 電気通信工事業（携帯基地局アンテナ・装機が主力）
- ポジション: **一次下請**（元請ではない）
- ビジョン: グループ100億円 / HSJ単体50億円を目指す
- 現状課題:
  - 入社時2週間の共通研修のみ、以降はOJT頼み
  - OJTメニューが未確立、教える側のスキルもバラバラ
  - 資格教育は社員ボランティア
  - 階層別の教育体系がない
  - 個々人の育成計画がない

---

## 1. 体系の全体構造

### 1-1. 5つの柱

| 柱 | 内容 | 対象 |
|---|---|---|
| **安全教育** | KY活動、リスクアセスメント、保護具、法令、事故事例 | 全社員（継続的） |
| **コア技術研修** | 光ファイバー、基地局施工、LAN配線、測定器、トラブルシューティング | 技術職中心 |
| **ヒューマンスキル研修** | コミュニケーション、報連相、プレゼン、ハラスメント防止 | 全社員（階層別） |
| **マネジメント研修** | リーダーシップ、工程管理、原価管理、コーチング、労務管理 | 課長代理以上 |
| **資格取得支援** | 階層別の必須・推奨資格、勉強会、受験費用補助 | 全社員 |

### 1-2. 階層区分（4職種×3階層）

| 階層 | 営業 | 技術 | 安品 | スタッフ |
|---|---|---|---|---|
| **課長・担当課長** | 営業課長 | 技術課長 | 安品課長 | 管理課長 |
| **課長代理** | 営業課長代理 | 技術課長代理 | 安品課長代理 | 管理課長代理 |

## 2. 求める人財像

各項目は「何ができるか（スキル）」で定義する。成果物や数値目標は評価制度側で設定する。

### 2-1. 営業職

※当社の営業職は技術知識を前提とした営業活動が求められる。技術を理解した上で客先と会話し、提案・調整を行う。

| スキル | 担当〜主任 | 課長代理 | 課長 |
|---|---|---|---|
| **技術理解** | 自社の施工内容・工法・使用機材を理解し、客先に説明できる | 客先の技術的な要望を理解し、社内の技術部門と正確に橋渡しできる | 技術動向を把握し、客先に対し技術的な裏付けを持った提案ができる |
| **信頼構築** | 期日・約束を守り、誠実な対応ができる | 客先の課題やニーズを把握し、的確に応えられる | 客先の経営課題を理解し、パートナーとして関係を築ける |
| **提案・交渉** | 見積依頼に対して正確に対応できる | 客先の要望を整理し、自社に有利な条件で調整できる | 価格・工期・仕様について客先と対等に交渉できる |
| **計数管理** | 実行予算の構造を理解し、予算内で業務を進められる | 見積作成から請求まで一連の数字を管理できる | 部門の売上・粗利を予測し、予実差異を分析できる |
| **案件管理** | 担当案件の工程・進捗を把握し、社内に共有できる | 案件の工程・コスト・リスクを総合的に判断し、社内外と調整できる | 複数案件を俯瞰し、優先順位とリソース配分を判断できる |
| **育成** | 自分の業務を言語化し、後輩に伝えられる | 後輩にOJTで体系的に指導できる | 部下の強み・課題を見極め、育成計画を設計できる |

| 4年目 | 提案力強化 | 客先ニーズを引き出し提案ができる | ルーティンは安定、提案型に移行中 | ヒアリング力、プレゼン力 | FP3級・簿記3級学習 |
| 5年目 | 主任準備 | 後輩のOJTができ、主任昇格準備完了 | 提案型営業が軌道に乗りつつある | 計数管理の視点、後輩指導 | 営業数字の自己管理 |

## 9. ブラザー・シスター制度

新入社員・中途入社者に対し、年齢の近い先輩社員を「ブラザー/シスター」として配置する。

| 項目 | 内容 |
|---|---|
| 対象 | 入社1〜2年目の社員 |
| ブラザー/シスターの条件 | 入社3〜7年目、同部門所属 |
| 担当人数 | 1名あたり新人1名 |
| 役割 | 実務指導 + 精神面のフォロー + 組織への馴染み支援 |
| 面談 | 週1回の定期面談（30分）+ 日常的なフォロー |
| 上長の関与 | 月1回、ブラザー/シスター・新人・上長の三者面談 |
| ブラザー/シスター研修 | 任命前に「指導者研修」（半日）を受講 |
| 期間 | 入社後1年間 |
